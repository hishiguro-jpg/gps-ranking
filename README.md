# GPS計測 順位表ジェネレーター

## ファイル構成

```
gps_ranking/
├── app.py           ← メインのアプリ（これを実行する）
├── requirements.txt ← 必要なライブラリのリスト
├── sample_data.csv  ← テスト用サンプルデータ
├── start.sh         ← 起動スクリプト（ダブルクリックでOK）
└── README.md        ← この説明書
```

## 初回セットアップ（最初の1回だけ）

ターミナルを開いて以下を順番に実行：

```bash
# 1. このフォルダに移動
cd ~/スポーツツーリズム_GPS/gps_ranking

# 2. ライブラリをインストール
pip install -r requirements.txt
```

## 起動方法

```bash
streamlit run app.py
```

ブラウザが自動的に開きます（http://localhost:8501）

## CSVの形式

| 列名 | 内容 | 例 |
|------|------|-----|
| 選手名 | 選手のフルネーム | 田中 太郎 |
| チーム名 | 所属チーム | チームA |
| タイム(分) | 完走タイム（分） | 52 |
| 距離(km) | 走行距離 | 10.5 |
| 最高速度(km/h) | GPSで計測した最高速度 | 18.2 |

※ 列名が少し違っても自動で認識します
※ 文字コードは UTF-8 / Shift_JIS どちらでもOK
